# banking-app

Banking App
Este projeto é um aplicativo bancário que oferece funcionalidades essenciais para gerenciamento de contas, transações, chaves PIX e autenticação de usuários.

  • Java
  
  • Spring Boot
  
  • Kafka
  
  • Elastic Stack
  
  • Docker
  
  • Kubernetes
